# Лексическая структура языка Astra

## Общие сведения

Исходный код программы на языке Astra представляет собой последовательность символов Unicode, закодированных в UTF-8.

Лексический анализ преобразует поток символов во входной поток лексем (токенов), используемый синтаксическим анализатором.

Лексер должен:
* игнорировать пробельные символы, кроме случаев, когда они разделяют лексемы;
* игнорировать комментарии;
* распознавать максимально длинные лексемы (правило maximal munch);
* генерировать специальную лексему `EOF` в конце входного потока.

Язык Astra чувствителен к регистру.

|return	  | возврат значения|
|if	      | условие|
|else	  | альтернативная ветка|
|while	  | цикл|
|break	  | выход из цикла|
|continue | следующая итерация|
|true	  | логическое значение|
|false	  | логическое значение|

## Операторы

Логические операторы представлены ключевыми словами and, or, not и символьными операторами &&, ||.

| Оператор | Предназначение               |
|----------|------------------------------|
| +        | Сложение                     |
| -        | Вычитание                    |
| *        | Умножение                    |
| /        | Деление                      |
| ==       | Сравнение "равно"            |
| !=       | Сравнение "не равно"         |
| <        | Сравнение "меньше"           |
| >        | Сравнение "больше"           | 
| <=       | Сравнение "меньше или равно" |
| >=       | Сравнение "больше или равно" |
| &&       | Логическое "и"               |
| \|\|     | Логическое "или"             |
| =        | Присваивание                 |
| ->       | Возвращаемый тип функции     |

```ebnf
ISO EBNF

operator =
      "+"
    | "-"
    | "*"
    | "/"
    | "&&"
    | "||"
    | "="
    | "=="
    | "!="
    | "<"
    | "<="
    | ">"
    | ">="
    | "->" ;
```

## Идентификаторы

Идентификатор — это последовательность английских букв (a...z, A...Z), арабских цифр (0...9) и подчёркиваний "_".

Идентификаторы используются для имен переменных, функций и других сущностей.

Правила:

* должны начинаться с буквы или `_`
* могут содержать буквы, цифры и `_`
* чувствительны к регистру

```ebnf
ISO EBNF

identifier =
    identifier_start ,
    { identifier_part } ;

identifier_start =
    letter
  | "_" ;

identifier_part =
    letter
  | digit
  | "_" ;

letter =
    "A"…"Z"
  | "a"…"z" ;

digit =
    "0"…"9" ;
```

## Литералы

### Числовые литералы

Astra поддерживает:
* целые числа
* числа с плавающей точкой

Знак не входит в литерал и обрабатывается как оператор.

Примеры:

```Astra
0
42
123456
3.14
0.5
10.0
```

```ebnf
ISO EBNF

integer_literal =
      "0"
    | non_zero_digit , { digit } ;

non_zero_digit =
    "1"…"9" ;

float_literal =
    digit ,
    { digit } ,
    "." ,
    digit ,
    { digit } ;
```

### Строковые литералы

Строковый литерал представляет собой последовательность символов, заключённую в двойные кавычки.

Примеры:

```Astra
"Hello"
"Astra"
"Hello, world"
""
```

### Escape-последовательности

| Вариант | Значение                                                                                     |
|---------|----------------------------------------------------------------------------------------------|
| \n     | Перенос строки                                                                                |
| \t     | Табуляция                                                                                     |
| \"    |  Двойная кавычка                                                                              |
| \\   | Обратный слэш                                                                                 | 
| \r     | Возврат каретки                                                                               |

Примеры:

```Astra
"Hello\nWorld"
"Quote: \"text\""
"Path: C:\\folder"
```
```EBNF
ISO EBNF

string_literal =
    '"' ,
    { string_character } ,
    '"' ;

string_character =
    normal_character
  | escape_sequence ;

escape_sequence =
    "\" ,
    ( "n" | "t" | "r" | "\"" | "\" ) ;

normal_character =
    ? любой Unicode символ кроме " и \ и перевода строки ? ;
```

## Пробелы и комментарии

### Пробельные символы

Следующие символы считаются пробельными:

*  `" "`
*  `"\t"`
*  `"\r"`
*  `"\n" `

Пробельные символы полностью игнорируются и не формируют лексемы.

### Однострочные комментарии

Начинаются с: `#` и продолжаются до конца строки.

### Многострочные комментарии

Начинаются с: `/*` и заканчиваются: `*/`

Вложенные комментарии запрещены.

```ebnf
ISO EBNF

comment =
    line_comment
  | block_comment ;

line_comment =
    "#" ,
    { ? любой символ кроме перевода строки ? } ;

block_comment =
    "/*" ,
    { ? любой символ кроме последовательности */ ? } ,
    "*/" ;
```
